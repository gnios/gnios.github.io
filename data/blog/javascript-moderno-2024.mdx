JavaScript evoluiu significativamente nos últimos anos, e dominar os recursos modernos é essencial para qualquer desenvolvedor. Vamos explorar as funcionalidades mais importantes do JavaScript ES6+ e como aplicá-las no dia a dia.

## 1. Arrow Functions

As arrow functions revolucionaram a sintaxe do JavaScript:

```javascript
// Sintaxe tradicional
function soma(a, b) {
  return a + b
}

// Arrow function
const soma = (a, b) => a + b

// Com um parâmetro (parênteses opcionais)
const quadrado = (x) => x * x

// Sem parâmetros
const saudacao = () => 'Olá, mundo!'
```

### Vantagens das Arrow Functions

- **Sintaxe concisa**
- **Lexical this** (não cria seu próprio contexto)
- **Ideal para callbacks**

```javascript
// Exemplo com array methods
const numeros = [1, 2, 3, 4, 5]

// Map com arrow function
const dobrados = numeros.map((n) => n * 2)

// Filter com arrow function
const pares = numeros.filter((n) => n % 2 === 0)

// Reduce com arrow function
const soma = numeros.reduce((acc, n) => acc + n, 0)
```

## 2. Destructuring

Destructuring permite extrair valores de arrays e objetos de forma elegante:

### Array Destructuring

```javascript
const cores = ['vermelho', 'verde', 'azul']

// Destructuring básico
const [primaria, secundaria, terciaria] = cores

// Pular elementos
const [primeira, , terceira] = cores

// Valores padrão
const [cor1, cor2, cor3, cor4 = 'amarelo'] = cores

// Rest operator
const [primeira, ...outras] = cores
```

### Object Destructuring

```javascript
const usuario = {
  nome: 'João',
  idade: 30,
  email: 'joao@email.com',
  endereco: {
    cidade: 'São Paulo',
    estado: 'SP',
  },
}

// Destructuring básico
const { nome, idade, email } = usuario

// Renomear variáveis
const { nome: nomeUsuario, idade: anos } = usuario

// Valores padrão
const { nome, idade, telefone = 'Não informado' } = usuario

// Destructuring aninhado
const {
  endereco: { cidade, estado },
} = usuario

// Rest operator em objetos
const { nome, ...outrosDados } = usuario
```

## 3. Template Literals

Template literals oferecem uma forma mais poderosa de trabalhar com strings:

```javascript
const nome = 'Maria'
const idade = 25

// Sintaxe tradicional
const mensagem = 'Olá, ' + nome + '! Você tem ' + idade + ' anos.'

// Template literal
const mensagem = `Olá, ${nome}! Você tem ${idade} anos.`

// Multilinhas
const html = `
  <div class="usuario">
    <h2>${nome}</h2>
    <p>Idade: ${idade}</p>
  </div>
`

// Expressões complexas
const preco = 100
const desconto = 10
const total = `Total: R$ ${preco - (preco * desconto) / 100}`
```

## 4. Spread Operator

O spread operator permite expandir arrays e objetos:

### Array Spread

```javascript
const frutas = ['maçã', 'banana']
const verduras = ['cenoura', 'alface']

// Combinar arrays
const alimentos = [...frutas, ...verduras]

// Adicionar elementos
const maisFrutas = ['laranja', ...frutas, 'uva']

// Copiar array
const frutasCopia = [...frutas]

// Converter NodeList para Array
const elementos = [...document.querySelectorAll('.item')]
```

### Object Spread

```javascript
const usuario = { nome: 'João', idade: 30 }
const endereco = { cidade: 'SP', estado: 'São Paulo' }

// Combinar objetos
const perfilCompleto = { ...usuario, ...endereco }

// Adicionar propriedades
const usuarioComEmail = { ...usuario, email: 'joao@email.com' }

// Sobrescrever propriedades
const usuarioAtualizado = { ...usuario, idade: 31 }
```

## 5. Promises e Async/Await

### Promises

```javascript
// Criando uma Promise
const buscarDados = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      const sucesso = Math.random() > 0.5
      if (sucesso) {
        resolve({ dados: 'Dados carregados!' })
      } else {
        reject(new Error('Erro ao carregar dados'))
      }
    }, 1000)
  })
}

// Usando Promise
buscarDados()
  .then((dados) => console.log(dados))
  .catch((erro) => console.error(erro))
```

### Async/Await

```javascript
// Função async
const carregarDados = async () => {
  try {
    const dados = await buscarDados()
    console.log(dados)
    return dados
  } catch (erro) {
    console.error('Erro:', erro.message)
    throw erro
  }
}

// Uso em IIFE
;(async () => {
  try {
    const resultado = await carregarDados()
    console.log('Sucesso:', resultado)
  } catch (erro) {
    console.error('Falha:', erro)
  }
})()
```

## 6. Modules (ES6)

### Export

```javascript
// named exports
export const PI = 3.14159
export const calcularArea = (raio) => PI * raio * raio

// default export
const Utilitarios = {
  formatarData: (data) => data.toLocaleDateString('pt-BR'),
  validarEmail: (email) => /^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email),
}

export default Utilitarios
```

### Import

```javascript
// named imports
import { PI, calcularArea } from './matematica.js'

// default import
import Utilitarios from './utilitarios.js'

// import tudo
import * as Matematica from './matematica.js'

// import dinâmico
const carregarModulo = async () => {
  const modulo = await import('./modulo-dinamico.js')
  return modulo
}
```

## 7. Classes

```javascript
class Animal {
  constructor(nome, especie) {
    this.nome = nome
    this.especie = especie
  }

  falar() {
    return `${this.nome} faz algum som`
  }

  // Getter
  get info() {
    return `${this.nome} é um ${this.especie}`
  }

  // Setter
  set novoNome(nome) {
    this.nome = nome
  }

  // Método estático
  static comparar(animal1, animal2) {
    return animal1.especie === animal2.especie
  }
}

class Cachorro extends Animal {
  constructor(nome, raca) {
    super(nome, 'cachorro')
    this.raca = raca
  }

  falar() {
    return `${this.nome} late: Au au!`
  }

  latir() {
    return 'Au au au!'
  }
}

// Uso
const rex = new Cachorro('Rex', 'Pastor Alemão')
console.log(rex.falar()) // "Rex late: Au au!"
console.log(rex.info) // "Rex é um cachorro"
```

## 8. Array Methods Modernos

```javascript
const produtos = [
  { nome: 'Notebook', preco: 2000, categoria: 'eletrônicos' },
  { nome: 'Mouse', preco: 50, categoria: 'eletrônicos' },
  { nome: 'Livro', preco: 30, categoria: 'livros' },
  { nome: 'Teclado', preco: 100, categoria: 'eletrônicos' },
]

// find - encontra o primeiro elemento
const notebook = produtos.find((p) => p.nome === 'Notebook')

// findIndex - encontra o índice
const indiceNotebook = produtos.findIndex((p) => p.nome === 'Notebook')

// some - verifica se pelo menos um elemento satisfaz a condição
const temEletronicos = produtos.some((p) => p.categoria === 'eletrônicos')

// every - verifica se todos os elementos satisfazem a condição
const todosTemPreco = produtos.every((p) => p.preco > 0)

// flatMap - mapeia e achata o resultado
const categorias = produtos.flatMap((p) => p.categoria.split(' '))

// reduce - reduz o array a um único valor
const totalEletronicos = produtos
  .filter((p) => p.categoria === 'eletrônicos')
  .reduce((total, p) => total + p.preco, 0)
```

## 9. Optional Chaining e Nullish Coalescing

```javascript
const usuario = {
  nome: 'João',
  endereco: {
    cidade: 'São Paulo',
    // telefone pode não existir
  },
}

// Optional chaining (?.)
const telefone = usuario.endereco?.telefone?.numero
const cidade = usuario.endereco?.cidade // 'São Paulo'

// Nullish coalescing (??)
const telefonePadrao = usuario.endereco?.telefone ?? 'Não informado'
const idade = usuario.idade ?? 0

// Combinando ambos
const infoCompleta = usuario.endereco?.telefone?.numero ?? 'Telefone não informado'
```

## 10. Dicas Práticas

### Performance

```javascript
// Use Map para operações frequentes de busca
const cache = new Map()
cache.set('chave', 'valor')
const valor = cache.get('chave')

// Use Set para valores únicos
const numerosUnicos = [...new Set([1, 2, 2, 3, 3, 4])]
```

### Debugging

```javascript
// Console.table para arrays de objetos
console.table(produtos)

// Console.group para organizar logs
console.group('Processamento de dados')
console.log('Passo 1: Carregando dados')
console.log('Passo 2: Processando dados')
console.groupEnd()
```
